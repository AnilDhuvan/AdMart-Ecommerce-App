# AdMart 🛒

AdMart is a modern e-commerce application built using:
- 💻 React (frontend)
- ☕ Spring Boot (backend)
- 🛢️ MySQL (database)

## Features
- Product listing, cart, and checkout
- User registration & login
- Admin dashboard (optional)
- Secure REST APIs
